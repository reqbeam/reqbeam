# Reqbeam Documentation

Welcome to the Reqbeam documentation. Reqbeam is a modern, open-source API testing and development platform that provides both a web interface and a powerful command-line tool.

## What is Reqbeam?

Reqbeam is a comprehensive API testing platform built with Next.js 15, TypeScript, Prisma, and Tailwind CSS. It provides everything you need to test, organize, and manage your APIs.

## Features

### Web Interface
- HTTP request builder with all methods
- Query parameters and headers management
- Multiple authentication types
- Response viewer with syntax highlighting
- Request history tracking
- Collections organization
- Environment variables
- Workspace management
- Mock servers

### CLI Tool
- Workspace management
- Environment management
- Request and collection management
- Request execution
- Test automation
- Logging and monitoring
- Real-time web UI sync
